title: Iterating matches
---

Object `Matcher` can be iterated, reading [`Detail`] in the process.

```php
$matcher = Pattern::of('\w+')->match('I like trains');

/**
 * @var Detail $match 
 */
foreach ($matcher as $match) {
    // use detail
}
```

Object [`Detail`] can be used to retrieve detailed information such as [`offset()`], [`index()`] etc.
